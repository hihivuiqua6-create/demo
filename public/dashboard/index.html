API Documentation
Hướng dẫn tích hợp vào web của bạn

Base URL

https://your-app.onrender.com

Tất cả request cần header: X-Api-Key: <your_api_key>

POST /api/create-bot Tạo bot mới

Tạo bot Telegram mới với token và URL web. Bot sẽ tự khởi động ngay.

Request Body:

{ "token": "1234567890:ABCDef...", // Bắt buộc "web_url": "https://yourwebsite.com", // Bắt buộc "welcome_message": "Xin chào! 👋", // Tùy chọn "button_text": "🚀 Mở App" // Tùy chọn }

Response:

{ "success": true, "bot": { "id": "uuid", "bot_username": "my_bot", "web_url": "https://yourwebsite.com", "telegram_link": "https://t.me/my_bot", "status": "active" } }
GET /api/bots Danh sách bot

Lấy toàn bộ danh sách bot đã tạo với trạng thái.

PUT /api/bot/:id Cập nhật bot
{ "web_url": "https://new-url.com", "welcome_message": "Chào mừng!", "button_text": "Mở App", "status": "active" // hoặc "inactive" }
GET /api/bot/:id/analytics Thống kê

Lấy thống kê người dùng và sự kiện của bot.

POST /api/bot/:id/broadcast Gửi tin nhắn hàng loạt
{ "message": "Tin nhắn gửi đến tất cả user" }
DEL /api/bot/:id Xóa bot

Dừng và xóa bot khỏi hệ thống.

📌 Ví dụ tích hợp JavaScript

// Thêm vào web của bạn const API_BASE = 'https://your-app.onrender.com'; const API_KEY = 'your_api_key_here'; async function createTelegramBot(botToken, webUrl) { const res = await fetch(`${API_BASE}/api/create-bot`, { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-Api-Key': API_KEY }, body: JSON.stringify({ token: botToken, web_url: webUrl, welcome_message: 'Xin chào! 👋 Nhấn nút để mở app.', button_text: '🚀 Mở Mini App' }) }); const data = await res.json(); if (data.success) { console.log('Bot created:', data.bot.telegram_link); } }
