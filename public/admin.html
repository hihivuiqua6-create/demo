📡 API Docs — Tích hợp vào web riêng

Base URL

https://your-app.onrender.com

🔑 API Key

Dùng trong tất cả request (giống admin password)

api_key: "admin123456"

➕ Thêm bot mới

POST /api/create-bot Content-Type: application/json { "token": "BOT_TOKEN_TỪ_BOTFATHER", "web_url": "https://your-shop.com", "welcome_message": "Chào mừng! 👋", "button_text": "🛒 Mở Shop", "api_key": "admin123456" } // Response: { "success": true, "bot_username": "your_bot", "message": "Bot @your_bot đã được kích hoạt!" }

♻️ Cập nhật bot

PUT /api/update-bot Content-Type: application/json { "token": "BOT_TOKEN", "web_url": "https://new-url.com", "api_key": "admin123456" }

🗑️ Xóa bot

DELETE /api/delete-bot Content-Type: application/json { "token": "BOT_TOKEN", "api_key": "admin123456" }

ℹ️ Kiểm tra trạng thái bot

GET /api/bot-status?token=BOT_TOKEN&api_key=admin123456 // Response: { "success": true, "bot": { "bot_username": "your_bot", "web_url": "https://your-shop.com", "is_active": 1, "is_running": true } }

💡 Ví dụ JavaScript tích hợp vào web riêng

// Hàm thêm bot từ form của bạn async function addTelegramBot(token, webUrl) { const res = await fetch('https://your-app.onrender.com/api/create-bot', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ token: token, web_url: webUrl, welcome_message: 'Chào mừng bạn! 👋', button_text: '🛒 Mở Shop', api_key: 'admin123456' // ← đổi nếu bạn đổi admin password }) }); const data = await res.json(); console.log(data); // { success: true, bot_username: "..." } }
