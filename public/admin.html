API & Hướng dẫn tích hợp

Gắn nền tảng vào website riêng của bạn.

1. Tạo bot từ web riêng

POST /api/create-bot

{
  "api_key": "<API_KEY>",
  "token": "123456:ABC...",
  "web_url": "https://yourapp.com",
  "welcome_message": "Chào {name} 👋",
  "button_text": "🛒 Mở Shop"
}

2. Biến thay thế trong tin nhắn chào

  • {name} — tên hiển thị (first_name hoặc username)
  • {first_name} — họ Telegram của user
  • {username} — @username của user (nếu có)
  • {full_name} — họ tên đầy đủ

Tất cả giá trị được HTML-escape để tránh injection khi parse_mode=HTML.

3. Bảo mật

  • Đặt env ADMIN_PASSWORD, JWT_SECRET, API_KEY trên production.
  • Login giới hạn 8 lần / 15 phút / IP. So sánh chuỗi dùng timingSafeEqual.
  • JWT 7 ngày. Lưu phía client trong localStorage — chỉ dùng HTTPS.
